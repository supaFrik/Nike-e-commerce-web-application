## 🚀 Quick Start Guide

### Prerequisites
Make sure you have these installed:
- ☕ **Java 11** or higher
- 🔧 **Maven 3.6+**
- 🌐 **Your favorite IDE** (IntelliJ IDEA, Eclipse, VS Code)

### Installation & Setup

1. **Clone the Repository**
   ```bash
   git clone https://github.com/supaFrik/Nike-e-commerce-web-application.git
   cd Javaweb32
   ```

2. **Build the Project**
   ```bash
   mvn clean install
   ```

3. **Run the Application**
   ```bash
   mvn spring-boot:run
   ```

4. **Open Your Browser**
   ```
   🌐 http://localhost:9090
   ```
